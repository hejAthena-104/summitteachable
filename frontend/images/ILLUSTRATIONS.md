# Summit Teachable — Illustration Guide

Storyset-style illustrations for the marketing site and in-app empty states.
Do **not** download random copyrighted art. Use [Storyset](https://storyset.com)
(free with attribution, or paid license to drop attribution), recolor to the
Summit Teachable purple, and export as SVG/PNG into this folder.

## Brand colors to apply

| Token        | Hex       | Use                                  |
|--------------|-----------|--------------------------------------|
| Purple      | `#8B5CF6` | Primary illustration accent / recolor |
| Lilac         | `#A78BFA` | Secondary highlights, gradients       |
| Purple base    | `#0F0A1E` | Background panels behind illustrations |
| Near-white   | `#ECEAF5` | Foreground text/details on dark        |

**Recommended Storyset customization color: `#8B5CF6`** (set this in the
Storyset editor "Customize colors" panel before downloading).

## Storyset URL pattern

```
https://storyset.com/<search-term>
# then open an illustration and use:
https://storyset.com/illustration/<slug>/<style>
# styles: amico | bro | rafiki | pana | cuate
```

In the editor: pick a style, click **Customize**, set the accent to `#8B5CF6`,
download as **SVG** (preferred) or transparent **PNG @ 2x**.

## Recommended search terms (4–5)

1. **`trading`** — markets / candlestick / stock-exchange scenes for the hero
   and "Markets" feature section. Suggested file: `illus-trading.svg`
2. **`online learning`** — courses / lessons / student-at-screen for the
   education pillar. Suggested file: `illus-learning.svg`
3. **`data analysis`** — charts / dashboards / analytics for the "Analytics"
   or portfolio-insights section. Suggested file: `illus-analysis.svg`
4. **`investing`** — growth / coins / portfolio for pricing or paper-trading
   value props. Suggested file: `illus-investing.svg`
5. **`finance`** *(optional)* — wallet / savings / growth chart for
   empty-states (e.g. "no trades yet"). Suggested file: `illus-empty.svg`

## Naming + sizing conventions

- SVG preferred; PNG fallback at 2x (e.g. 1200px wide for hero, 640px for sections).
- Transparent background so they sit on navy panels.
- Keep filenames `illus-<topic>.svg` to match the two hand-authored samples below.
